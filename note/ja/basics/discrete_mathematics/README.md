# 『離散数学』ノート


## 概要

**離散数学**とは有限で離散的な値を扱う数学のうち、非計算の分野を対象としたもの。[コンピュータ](../../computer/_/chapters/basic_knowledge_of_computer.md#コンピュータ)では主に離散値を扱うため、離散数学の考え方が基本となる。
