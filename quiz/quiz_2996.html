PHPは、動的なWebサイトを作成するためのツールから派生したスクリプト言語。Web開発に特化しており、HTMLに埋め込むような記法を用いることができるという特徴がある。そのほかにも、学習コストが低い点やデータベースアクセスを容易に行えるという強みもある。

<?php

$input_num = $_POST["input_num"];

if( $input_num % 2 === 0 )
{
    echo("It is an even number : " . $input_num);
}
else
{
    echo("It is an odd number : " . $input_num);
}

?>
